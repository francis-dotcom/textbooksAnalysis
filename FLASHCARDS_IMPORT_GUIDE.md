## üì• How to Import into Anki

### Method 1: Anki Desktop

1. **Open Anki** on your computer
2. Click **File ‚Üí Import**
3. Select the CSV file (e.g., `Chapter_1_Flashcards.csv`)
4. In the import dialog:
   - **Type**: Basic
   - **Deck**: Choose or create a deck (e.g., "International Finance - Chapter 1")
   - **Fields separated by**: Comma
   - **Allow HTML in fields**: ‚úì (IMPORTANT - enables formatting)
   - **Field 1**: Front
   - **Field 2**: Back
   - **Field 3**: Tags
5. Click **Import**
6. Repeat for other chapters

### Method 2: AnkiWeb

1. Go to [ankiweb.net](https://ankiweb.net)
2. Log in to your account
3. Create a new deck
4. Use the desktop app to import (AnkiWeb doesn't support direct CSV import)
5. Sync to access on mobile

## üìñ Study Strategies

### 1. **First Pass - By Chapter**
Study one chapter at a time to build foundational knowledge:
```
Day 1-2: Chapter 1 (20 cards)
Day 3-4: Chapter 3 (30 cards)  
Day 5-7: Chapter 5 (47 cards)
```

### 2. **Second Pass - By Type**
Focus on weak areas:
- Filter by `Literal` + `Definition` for quick recall
- Filter by `Conceptual` for deeper understanding
- Filter by `Applied` + `Calculation` for problem-solving

### 3. **Pre-Test Review**
- Review all `Applied` and `Calculation` cards
- Focus on cards you've marked as "Hard" or "Again"
- Do a full review of all chapters

### 4. **Active Recall Tips**
- **Don't flip too quickly** - Try to fully answer before revealing
- **Say answers out loud** - Reinforces memory
- **Write calculations** - For math problems, work them out on paper
- **Mark difficult cards** - Use Anki's "Hard" button to review more frequently

## üí° Pro Tips for Your Test

### 1. **Focus on Calculations**
The applied cards with calculations are most likely to appear on tests:
- Cross-rate calculations (Chapter 5)
- BOP identity problems (Chapter 3)
- Forward premium/discount (Chapter 5)

### 2. **Understand Relationships**
Don't just memorize - understand WHY:
- Why does BOP sum to zero?
- How does currency depreciation affect trade?
- Why does triangular arbitrage work?

### 3. **Practice with Variations**
For calculation cards, try:
- Changing the numbers
- Working backwards
- Explaining the process to someone else

### 4. **Use Spaced Repetition**
- Start studying NOW, not the night before
- Review cards daily, even if just 15 minutes
- Trust Anki's algorithm - it knows when you need to review

### 5. **Create Mental Models**
- Visualize the BOP accounts as a balance sheet
- Draw diagrams for triangular arbitrage
- Create acronyms for lists (e.g., "CCFI" for Current, Capital, Financial, International reserves)
